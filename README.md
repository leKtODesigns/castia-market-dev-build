# Castia Market — Price Tracker

A non-commercial fan-made GitHub Pages site that displays live Castia auction-house price data for player guidance. It does not broker real-money trades or item sales.

## Features

- Live price data from the Castia Worker API
- Search by item name or category
- Filter by category and confidence level
- Sort by any column (price, samples, trend, confidence)
- Paginated results (50 per page)
- Auto-refreshes every 5 minutes
