Privacy

What the site connects to

Castia Market is a static website. It does not operate user accounts, checkout flows, or real-money transactions.

Third-party services used

  • GitHub Pages hosts the website files.
  • Google Fonts serves the font files used by the interface.
  • The Castia Worker API serves public market data used by the listings and sellers views.
  • Crafty serves external Minecraft avatar renders shown on seller cards and panels.

Local browser storage

The site may store interface preferences such as view mode, favorites, compare selections, or similar local state in the browser so the UI behaves consistently between visits.

No commerce

Castia Market is informational only. It does not sell items, facilitate real-money trading, or process payment details.

Castia Market

Castia Market is an unofficial fan-made price tracker. It is not approved by, endorsed by, or associated with Mojang, Microsoft, CastiaMC, or Crafty.
