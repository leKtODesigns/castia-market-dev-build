Asset credits and provenance

Castia Market uses Minecraft-related artwork, Castia-related artwork, one third-party resource-pack set, and external avatar rendering. This page gives the human-readable attribution summary; the detailed file inventory is linked below.

Minecraft and Mojang assets

Minecraft names, textures, screenshots, and official game assets remain owned by Mojang Studios and Microsoft. Castia Market is an unofficial fan-made site and uses Minecraft-related imagery only to identify in-game items for price-tracking purposes.

Source file pages for locally hosted Minecraft-related item images are recorded in the manifest where known. These assets are reviewed against the Minecraft Usage Guidelines.

Castia-related assets

Some server-specific item artwork originates from CastiaMC or the Official CastiaMC Wiki. The CastiaMC Wiki footer states that page content is under Creative Commons Attribution-ShareAlike 4.0 unless otherwise noted; individual asset status is still tracked conservatively in the manifest when a file page does not provide a separate clear license statement.

Castia Market is not approved by, endorsed by, or associated with CastiaMC.

Enchanted book textures

Enchanted book textures are from Re-covered by carl-classic, which states a CC BY-NC-ND 4.0 license for the project artwork.
